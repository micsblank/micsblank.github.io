Set Game

Instructions

Select three cards that form a set.

A set consists of three cards in which each of the card's attributes (hue, saturation, and lightness) when looked at separately are either all the same or all different.

Found sets

This is an automatic composition generator, made to show how hue, saturation and lightness change the weight and feel of a piece visually. I made this to use as inspiration to gain a better understanding of colour theory.
